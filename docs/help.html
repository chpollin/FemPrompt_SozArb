Hilfe

Wissens-Chat

Stellen Sie Fragen an das synthetisierte Wissen aus 326 Papers. Die Antworten werden von Gemini 3 Flash generiert, mit Kontext aus dem Forschungskorpus.

Wichtig: Die Antworten sind zweifach LLM-vermittelt -- einmal durch die Wissensextraktion (Claude Haiku 4.5), einmal durch die Antwortgenerierung (Gemini 3 Flash). Inline-Zitationen (z.B. Zakharova et al. (2024)) sind klickbar und fuehren direkt zum Paper im Korpus zur Verifikation.

Sie benoetigen einen kostenlosen Google AI API Key.

Wissensnetz

136 Konzepte, extrahiert aus 249 Wissensdokumenten. Groesse = Haeufigkeit im Korpus. Verbindungslinien = gemeinsame Papers. Klick auf ein Konzept zeigt das Ego-Netzwerk (Nachbarn hervorgehoben, Rest abgeblendet) mit Definition und Paper-Liste.

  • Farben: Technik, Sozial, Bruecke
  • Gestrichelte Ringe = Konzepte in Divergenz-Faellen
  • Frequenz-Slider filtert nach Mindesthaeufigkeit

Bewertungsvergleich

Vergleich der LLM- und Expert:innen-Bewertungen. Slope Chart zeigt die Kategorie-Divergenz, Konfusionsmatrix die Uebereinstimmung. Kappa-Werte pro Kategorie. Klick auf eine Konfusionsmatrix-Zelle filtert den Korpus.

Korpus

Alle 326 Papers als durchsuchbare Tabelle. Filter nach LLM/Human-Entscheidung, Jahr, Kategorien. Klick auf eine Zeile oeffnet das Detail-Panel mit Assessment-Vergleich, Abstract und LLM-Begruendung. Der Korpus ist die Referenzschicht -- von allen anderen Bereichen aus erreichbar ueber Zitationslinks.
