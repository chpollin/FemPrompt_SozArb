# SocialAI Projekt - Meeting-Notizen

## Projektübersicht

**Kooperation:** LIST und Laura Bassi Programm  
**Fokus:** Bias und Fairness in generativer KI durch Prompting

## Überschneidungen LIST / Laura Bassi

- Gemeinsamer Literaturreview zu Bias und Fairness
- Auswertungs-, Interpretations- und Publikationsphase
- LIST Methodenpaper

## Konkreter Workflow

### Literaturverarbeitung

```
Zotero → Python → Tabellen → Obsidian → Validierung → Zotero
```

### Automatisierung geplant

1. **Deep Research** → RIS-Export (mit Death/Summary)
2. **Zotero**: Human-in-the-Loop Collection
3. **Python**: Tabellen für Bewertung erzeugen
4. **Obsidian**: Netzwerk-Visualisierung
