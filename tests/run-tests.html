PRISM pure-function tests (plan P1, zero-dependency browser runner)

Open this file directly over file:// or from any static server; no build step, no framework. All fixtures are inline in tests.js. The only network attempt is prisma-data.js trying to load the research vault, which fails here and is caught; the tests do not depend on it. Results render below, into document.title (PASS n/n or FAIL k/n), and into window.__TEST_RESULTS__, so a browser agent can read the outcome from the title alone. Expected agreement values come from knowledge/verification-empirical-core.md.
