# Transformative Justice Framework

## Definition

Ein Gerechtigkeitsansatz, der nicht nur Täter-Opfer-Beziehungen adressiert, sondern die strukturellen Bedingungen reformiert, die Schäden ermöglicht haben, mit Fokus auf Prävention zukünftiger Harms.
