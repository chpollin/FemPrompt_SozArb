# Disagreement Cases

Papers, bei denen Human Expert und Agent Workflow unterschiedliche
Entscheidungen getroffen haben.

## Für qualitative Analyse

Diese Fälle sind besonders interessant für die Paper-Analyse:
- Wo übersieht der Agent relevante Papers?
- Wo sieht der Agent Relevanz, die Expert:innen nicht sehen?
- Welche Kategorien sind problematisch?

## Kategorien

### Human Include, Agent Exclude (Kritisch)
- Agent übersieht relevante Papers

### Human Exclude, Agent Include
- Agent sieht Relevanz, die Expert:innen nicht sehen

### Unclear Cases
- Fälle mit "Unclear" Bewertung
